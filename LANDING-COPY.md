## How it works

1. Buy the harness. You get access to the GitHub repo and a commercial license key.
2. Clone, `npm install`, drop in your Anthropic API key, point it at your site directory.
3. Open `localhost`, start chatting. The agent edits files; the preview refreshes.

## FAQ

**Can I try it before buying?**
Yes. The repo is public. Clone it, run it, evaluate it for thirty days. If you decide to use it commercially, buy the license. Honor system.

**Can I use it for client work / inside my company?**
Yes — that's exactly what the $99 license is for. One license per person or per legal entity. If you're not sure which applies, email operator@promptish.io and we'll sort it out.

**Do I get updates?**
Not as part of v1. $99 buys the current state of the harness. We may release v2 as a free upgrade, a paid upgrade, or not at all. Buy assuming you're getting today's version forever.

**What about support?**
GitHub Issues, no SLA, best effort. We read every issue. We fix bugs we can repro. We don't promise turnaround times. If you need contracted support, that's a separate conversation.

**Can I redistribute it / fork it / sell it?**
No. You can modify it for your own use. You can submit pull requests. You can't host it as a service for others, repackage it, or resell it.

**What do I need to run it?**
Node.js 20+, an Anthropic API key (you pay Anthropic separately for usage), and the Claude Code CLI installed locally. Mac, Linux, or Windows. Roughly 200 MB of disk after `npm install`.

**How big is it?**
Small. Under 5,000 lines of code. You can read the whole thing in an evening. That's intentional — you should be able to audit and modify it without hiring us.

**Is this a SaaS / hosted product?**
No. Self-hosted only. Runs on your machine. Your API key, your files, your control.
